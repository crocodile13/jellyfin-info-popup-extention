# assets/

Ce dossier contient les ressources statiques du plugin.

## icon.png

Icône du plugin affichée dans le catalogue Jellyfin.
- Taille recommandée : **256×256 px**
- Format : PNG avec transparence
- Référencée dans `manifest.json` → `imageUrl`

**Remplacez ce fichier par votre propre icône avant de publier.**
